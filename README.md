# AssettoSync Client

Client for AssettoSync - app for downloading and installing mods to Assetto Corsa game.

## Table of contents

* [Usage](#usage)
* [Sceenshots](#screenshots)

## Usage

```
git clone https://github.com/muttleyxd/assettosync_client.git
cargo run
```
